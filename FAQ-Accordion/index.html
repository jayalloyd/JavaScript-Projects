Frequently Asked Questions

The **Virtual DOM (VDOM)** is a programming concept where a virtual representation of the UI is kept in memory and synced with the "Real" DOM by a library like React. This allows React to figure out the minimal set of changes needed and update the browser efficiently, rather than re-rendering the entire page.

This is the **Immutability rule**. If you directly change (mutate) an array or object in React, the VDOM cannot detect that a change occurred because the object reference remains the same. You must use methods like `.map()` or `.filter()` to return a **new array reference**, which signals to React that a re-render is required.

Flexbox is used for **arranging, aligning, and distributing space** among items *inside* a container. It's necessary when you need to center content (horizontally and vertically) or create responsive navigation layouts.
